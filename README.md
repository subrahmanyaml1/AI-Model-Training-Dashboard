# 🤖 AI Model Training Dashboard

A professional Streamlit-based Machine Learning dashboard that allows users to upload datasets, preprocess data, train ML models, and make predictions interactively.

## 🚀 Features
* Upload CSV dataset
* Automatic preprocessing
* Remove null values and duplicates
* Manual column removal
* Automatic problem type detection (Classification / Regression)
* Multiple ML algorithms support
* Model training with Train/Test score
* Prediction using manual input
* Professional dark dashboard UI

## 🛠 Technologies Used
* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn

## 📂 Supported Algorithms

### Classification
* KNN
* Naive Bayes
* Decision Tree
* Logistic Regression
* SVM

### Regression
* KNN Regressor
* Linear Regression
* Decision Tree Regressor

```

```bash
streamlit run app.py

## 📌 Project Workflow
1. Upload dataset
2. Select target column
3. Preprocess automatically
4. Choose algorithm
5. Train model
6. Predict output
